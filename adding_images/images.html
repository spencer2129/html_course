HTML Images

We'll discuss how to use the <img> element, and annotating with captions using <figure>.

Putting an image on a webpage

In order to put an image on a web page, you must use the <img> element. This is a void element which means that it cannot have a child content and cannot have an end tag. This element requires two attributes to be useful src and alt. The src attribute contains the URL pointing to the image you want to embed in the page, and can be a realtive URL or an absolute URL. Without the src attribute, an img element has no image to load. The alt attribute is supposed to contain a textual description of the image, for use in situations where the image cannot be seen/displayed or takes a long time to render because of slow connections. Alt text is useful when the user is visually impared and is using a screen reader to read the web, or when a browser does not support the image type, or to provide text for search engines to utilize.

<img src="./images/boracay.jpg" alt="Boracay, Philippines" />

You can also embed the image using its absolute URL

<img src="https://www.example.com/images/boracay.jpg" alt="Boracay, Philippines" />

Alternative text

Use alternate text alt to give a textual description of the image

<img 
                src="./images/boracay.jpg"
                alt="This is White Beach in Boracay, Philippines" />

Width and height

Use width and height attributes to specify the width and height of your image. These attributes only take integer values without units, such as px.

<img 
                src="./images/boracay.jpg"
                alt="This is White Beach in Boracay, Philippines"
                width="400"
                height="350" />
