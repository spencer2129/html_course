The Head Element

In this small lesson we learn about the head element, and the elements inside such as the meta, title, and link elements.

The <head> element is a container for metadata and it is placed between the <html> tag and the <body> tag. Metadata is data about the HTML document; it is not displayed and it typically defines the document title, character set, styles, scripts and other meta information. There can only be one <head> in an HTML document.
