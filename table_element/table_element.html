HTML Table Element

The <table> HTMl element represents information presented in a two dimensional table comprised of rows and columns of cells containing data.

Layout of Table Contents

Definitions

<caption>
The <caption> HTML element specifies the caption or title of a table, providing the table an accessible description
<thead>
The <thead> HTML element encapsulates a set of table rows, indicating that they comprise the head of a table with information about the table's columns.
<colgroup>
The <colgroup> HTML element defines a group of columns within a <table>. This should appear after the <caption> element, but before any <thead>, <tbody>, <tfoot> and <tr> elements
<col>
The <col> HTML element defines one or more columns in a column group represented by its parent <colgroup> element. The <col> element is only valid as a child of a <colgroup> element that has no <span> attribute defined.
<th>
The <th> HTMl element defines a cell as the header of a group of table cells and may be used a child of the <tr> element. The exact nature of this group is defined by the scope and headers attributes. Usually the first row (<tr> element) contains the columns headers (<th> elements), which act as the "titles" for the columns to make it easier to understand the information in the columns and identify the data.
<tbody>
The <tbody> HTML element encapsulates a set of table rows (<tr> elements), indicating that they comprise the body of a table's main data.
<tr>
The <tr> HTML element defines a row of cells in a table. The row's cells can then be established using a mix of <td> (data cell) and <th> (header cell) elements.
<td>
The <td> HTML element defines a cell of a table that contains data and may be used as a child of the <tr> element.
<tfoot>
The <tfoot> HTML element encapsulates a set of table rows (<tr> elements), indicating that they comprise the foot of a table with information about the table's columns. This is usually a summary of the columns, such as, a sum of the given numbers in a column.

Example

Accessibility

Captions

The <caption> element gives a clear description of the table's purpose, and it helps the user decide whether to check the rest of the table or not. caption helps those individuals using the aid of assistive technology.

Scoping rows and columns

The <scope> value on the header cells (<th> elements) is reduntant, however some assistive technologies may fail to draw correct inferences, so specifying header scope may improve user experiences.

Complicated tables

Assistive technologies such as screen readers may have difficulties parsing tables that are complex that header cells can't be associated in a strictly horizontal or vertical way, which can happen when you use <colspan> and <rowspan> attributes.
