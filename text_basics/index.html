Text Basics

Much of the text content on a web page will be made of headings and paragraphs. Headings have a hiearchy. There can only be one <h1> in the page. After that they can have subtopics using <h2> elements. The subtopics can have subtopics using <h3> and so on.

The headers give semantic meaning. Screen readers can also navigate your page better if you're using headers to convey that this section is a subtopic of the main header, and so on.
